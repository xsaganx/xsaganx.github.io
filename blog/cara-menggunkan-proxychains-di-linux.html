cara neggunakan proxychains di linux

ini adalah tutorial cara menyeting dan menggunakan proxychain pada linux, kali ini saya akan melakukannya pada kali linux.

- install tor

salah satu kegagalan saat menjalankan proxychains adalah dikarnakan tor pada linux belum terinstal, oleh karena itu kita harus mengisntal tor terlebih dahulu, caranya cukup mudah.

ketikan pada terminal: sudo apt install tor

- mencari dan mengedit file proxychains.conf

setelah itu cari file proxychains dan mengedit cara ketikan :

locate proxychains

setelah itu buka file proxychains.conf atau juga ada yang proxychains4.conf itu sama saja. buka menggunakan nano. ketikan

sudo nano etc/app/proxychains.conf
